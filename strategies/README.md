# Strategies (learning exercises)

Place practice algorithms in this directory. Each strategy should subclass
`hedgekit.strategy.Strategy` and implement `on_bars()`.

**This is not financial advice.** Code you add here is for **education**. Test in
**simulated** mode. Read [docs/EDUCATION.md](../docs/EDUCATION.md) and follow
[docs/LEARNING_PATH.md](../docs/LEARNING_PATH.md). **Do not use real money**
while learning.

Copy `example_mean_reversion/` as a starting template.
